Welcome to Wireguard VPN

Server configuration

All server configuration is done in CLI. Please log in via SSH or USB console

Client configuration

You can grab client configuration in the /clients-section.

Adding new clients

New clients can be added from the cli, using the command sudo ./new_vpn_user.sh <user> 

usr@wireguard:~ $ sudo ./new_vpn_user.sh Alice

Removing users

Removing user must be done from ssh cli, using command sudo pivpn remove <username>
